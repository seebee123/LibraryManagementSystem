# 📚 Library Management System (ASP.NET Core MVC)

A full-featured **Library Management System** built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.  
This project helps manage books, students, issue/return operations with fine calculation, pagination, and Excel export.

---

## 🚀 Features

### 👨‍🎓 Student Module
- Add / Edit / Delete Students
- Search Students (Name, Roll No, Department, Email)
- Pagination support
- Export Students data to Excel

### 📚 Book Module
- Add / Edit / Delete Books
- Search Books (Title, Author, ISBN, Category)
- Pagination support
- Export Books data to Excel

### 📖 Issue Book Module
- Issue books to students
- Return books system
- Fine calculation for late return
- Track issued & returned books
- Search + Pagination
- Export issued records to Excel

### 📊 Dashboard
- Total Books
- Total Students
- Issued Books count
- Returned Books count
- Total Fine collected

---

## 🛠️ Tech Stack

- ASP.NET Core MVC (.NET)
- Entity Framework Core
- SQL Server
- Bootstrap 5
- ClosedXML (Excel Export)
- HTML / CSS / JavaScript
